# Controle Bot 🤖📲

Bot automatizado para atendimento via **WhatsApp**, desenvolvido com o objetivo de responder mensagens, organizar agendamentos e facilitar a comunicação com clientes de forma simples e eficaz.

---

## 🚀 Tecnologias utilizadas

* **Node.js**
* **whatsapp-web.js**
* **qrcode-terminal**
* **fs (File System)**
* **JavaScript**

---

## 🧠 Funcionalidades

* Leitura de mensagens recebidas no WhatsApp
* Respostas automáticas baseadas em palavras-chave
* Envio de menus com opções para o usuário
* Controle de respostas e interação por etapas
* Registro de interações em arquivos locais (`.json`)
